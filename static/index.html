quien.sh
— what you get
> whois

RDAP + WHOIS

RDAP-first lookups with automatic WHOIS fallback. IANA referral for any TLD.

> dns

DNS Records

Full record resolution. Nameservers, A/AAAA, MX, TXT, and more.

> mail

Mail Audit

MX, SPF, DMARC, DKIM, and BIMI validation with VMC chain checks.

> tls

TLS / SSL

Certificate details, chain validation, expiry dates, and cipher info.

> stack

Tech Stack

Detect frameworks, CMS, CDN, analytics, and third-party services from HTML.

> seo

SEO Analysis

Indexability, structured data, Open Graph, meta tags, and Core Web Vitals.

> http

HTTP Headers

Response headers, security headers, redirects, and server info.

> 8.8.8.8

IP Lookup

Reverse DNS, ASN discovery via RDAP, BGP fallback, and PeeringDB enrichment.

> all --json

JSON Output

Every subcommand outputs structured JSON for scripting and automation.

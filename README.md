# quien.sh

Try [quien](https://github.com/retlehs/quien) instantly over SSH — no install required.

```
ssh quien.sh
```

## What is quien?

quien is an interactive WHOIS lookup tool with tabbed views for WHOIS, DNS, mail configuration, SSL/TLS, HTTP headers, SEO analysis, and tech stack detection.

This SSH service drops you into the interactive prompt so you can try it out before installing. For full functionality, [see the quien repo](https://github.com/retlehs/quien) for install options and documentation.
